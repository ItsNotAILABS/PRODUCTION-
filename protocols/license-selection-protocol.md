# License Selection Protocol

Default posture: maximum ownership control.

## Do Not Add

Do not add any of the following unless explicitly instructed:

- MIT
- Apache
- GPL
- LGPL
- AGPL
- BSD
- CC0
- CC BY
- CC BY-SA
- CC BY-NC-SA

## Default Rights Text

Copyright © Alfredo Medina Hernandez.

All rights reserved unless otherwise stated.

Public visibility does not grant reuse rights.

## Platform-Specific Preference

Zenodo: choose the most restrictive available rights option.  
OSF: use no license where permitted.  
arXiv: use arXiv.org perpetual, non-exclusive license.
