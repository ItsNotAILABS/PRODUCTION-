# Zenodo Release Protocol

Zenodo is the primary release route for MedinaSITech research papers when arXiv is blocked, delayed, or strategically unnecessary.

## Use Zenodo When

- A DOI is needed.
- Public timestamped release is needed.
- arXiv endorsement is unavailable.
- The work should become citable.
- The release should preserve source files and PDFs.

## Steps

1. Log in to Zenodo.
2. Create a new upload.
3. Upload the clean PDF.
4. Upload the source zip if desired.
5. Set resource type to Publication / Preprint.
6. Add title, author, affiliation, abstract, and keywords.
7. Choose the most restrictive rights/license option available.
8. Reserve or register DOI.
9. Preview carefully.
10. Publish only when final.
11. Copy DOI and URL into `RELEASE_LEDGER.md`.
12. Add a record under `zenodo/records/`.

## Rule

Treat Zenodo publish as archival. Material changes should create a new version.
