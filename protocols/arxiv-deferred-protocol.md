# arXiv Deferred Protocol

arXiv is a deferred route, not a blocker.

## Current Position

If arXiv requires endorsement and endorsement is unavailable, do not stall the research release.

Use:

1. Zenodo first.
2. OSF second if needed.
3. arXiv later.

## arXiv License Choice

Use:

arXiv.org perpetual, non-exclusive license

Avoid:

- CC0
- CC BY
- CC BY-SA
- CC BY-NC-SA

Use CC BY-NC-ND only if specifically instructed.
