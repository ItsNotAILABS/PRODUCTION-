# OSF Release Protocol

OSF is the secondary route for public preprint visibility and author-controlled sharing.

## Use OSF When

- A public preprint page is needed.
- No arXiv endorsement is available.
- Maximum rights control is preferred.
- A persistent URL is sufficient.

## Steps

1. Log in to OSF.
2. Create a preprint or project record.
3. Upload the clean PDF.
4. Add source zip if desired.
5. Enter metadata.
6. Choose no license where permitted if maximum copyright control is desired.
7. Publish or submit.
8. Copy URL into `RELEASE_LEDGER.md`.
9. Add record under `osf/records/`.
