# MedinaSITech Research

This repository contains MedinaSITech research preprints, architecture artifacts, protocols, release records, and supporting source materials.

The current research line includes:

1. **De Superficie Terminorum: A Native Workstation Surface for Governed Agentic Execution**
2. **Ratio Ordinis: A Risk-Aware AI Routing Algorithm for Paradigm-Aware Agentic Execution**
3. **Machina Polyglotta Ordinata: Multi-Terminal AI Orchestration for Cost-Aware Polyglot Computing**

This repository is public or shareable for reading, citation, provenance, and research visibility only. Public visibility does not grant reuse rights.

See `RIGHTS.md`.

## Repository Structure

- `papers/` — released or release-ready papers, PDFs, source, metadata, and notes.
- `specs/` — architecture artifacts and system specifications.
- `protocols/` — repeatable release, licensing, Zenodo, OSF, arXiv, and GitHub coordination protocols.
- `zenodo/records/` — DOI and release records.
- `osf/records/` — OSF release records, if used.
- `arxiv/deferred/` — arXiv materials held for later submission.
- `archive/` — drafts, old exports, superseded material, and triage files.

## Operating Rule

Do not stall on arXiv endorsement.

Primary release route: Zenodo.  
Secondary route: OSF.  
Deferred route: arXiv.  

Release the work cleanly. Preserve ownership. Keep the next route open.
